Sorting Algorithms A brief overview of the more common sorting algorithms. Binary search was efficient when given a pre-sorted data set, but what are the most efficient ways to actually sort data sets? There are many different sorting algorithms, but you are going to focus on three of the most common: bubble sort, merge sort, and quicksort. Each has pros and cons when it comes to their efficiency, as you can see in the table below: Best Case Worst Case Average Case Space Complexity Bubble Sort Ω(n) O(n^2) O(n^2) O(1) Merge Sort Ω(n log n) O(n log n) O(n log n) O(n) Quicksort Ω(n log n) O(n log n) O(n^2) O(log n)
